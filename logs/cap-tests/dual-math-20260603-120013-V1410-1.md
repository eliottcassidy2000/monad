## Autonomous math session — tournament theory

**Question I chose:** What is the *expected number of Hamiltonian (directed) paths* in a uniformly random tournament on $n$ labeled vertices?

**Setup.** A tournament on $n$ vertices is an orientation of $K_n$: for each unordered pair $\{i,j\}$, exactly one of $i\!\to\!j$ or $j\!\to\!i$ is chosen. There are $\binom{n}{2}$ pairs, so $2^{\binom n2}$ tournaments, each equally likely. A *Hamiltonian path* is an ordering $v_1,\dots,v_n$ of all vertices with every consecutive arc oriented forward: $v_1\!\to\!v_2\!\to\!\cdots\!\to\!v_n$.

**Derivation (first-principles, linearity of expectation).**
For a fixed permutation $\sigma=(v_1,\dots,v_n)$ of the vertices, define indicator $X_\sigma = 1$ if $\sigma$ is a Hamiltonian path. This requires the $n-1$ consecutive arcs to each point the right way. These $n-1$ arcs involve $n-1$ *distinct* pairs, and each pair is oriented independently with probability $\tfrac12$ in the right direction. Hence
$$\mathbb{E}[X_\sigma] = \left(\tfrac12\right)^{n-1}.$$
The total count of Hamiltonian paths is $X=\sum_\sigma X_\sigma$ over all $n!$ permutations. By linearity,
$$\boxed{\;\mathbb{E}[X] = \frac{n!}{2^{\,n-1}}\;}$$

**Verification by full enumeration, $n=3$.** Formula gives $6/4 = 1.5$. The $8$ tournaments split into:
- $6$ transitive ones (e.g. $a\!\to\!b\!\to\!c,\ a\!\to\!c$): exactly **1** Hamiltonian path each (only $abc$ works).
- $2$ cyclic ones (3-cycles): exactly **3** paths each ($abc,bca,cab$ all close up).

Average $=\dfrac{6\cdot 1 + 2\cdot 3}{8}=\dfrac{12}{8}=1.5.$ ✓ Matches.

Also $n=2$: formula $2/2 = 1$; every 2-vertex tournament is a single arc with exactly 1 path. ✓

**Remarks.** The expectation grows like $n!/2^{n-1}$ (super-exponentially), yet Rédei's theorem guarantees *every individual* tournament has at least one — indeed an **odd** number — of Hamiltonian paths, consistent with both enumerated counts above (1 and 3 are odd, and the cyclic case's 3 illustrates the odd-count theorem nicely).

RESULT: The expected number of directed Hamiltonian paths in a uniformly random labeled tournament on $n$ vertices is exactly $n!/2^{n-1}$, verified by full enumeration at $n=2,3$.
CONFIDENCE: high — clean linearity-of-expectation argument (consecutive arcs use distinct independent pairs) confirmed against exhaustive hand-enumeration for $n=3$.
